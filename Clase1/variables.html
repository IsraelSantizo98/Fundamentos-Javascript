Variables en Javascript

Las variables tiene 3 grupos grandes los cuales son:

  • var: Es una variables global
  • const: Es una variable que no se puede cambiar su contendido como una variable de la velocidad de la luz
  • let: Es una variable que permite declarar limitando su alcance

Convencion

CamelCase

Declarar variables

Para declar una variable string se asigna que tipo de variable sera, luego se coloca el nombre de la variable y posteriormente se asigan su valor en el caso de string se debe colocar comillas simple ' ' o comillas dobles " ".

var = nombre;
nombre = 'Israel';
Por convencion se recomienda usar comillas simples ' '.

El punto y coma ; es algo opcional ya que reconoce las lineas de codigo

Funcion basicas

Las funciones se reconocen por que al final llevan parentesis

  • Mayusculas o minusculas

    • var nombreMayusculas = nombre.toUpperCase()

    • var apellidoMinusculas = apelllido.toLoweCase()

  • Saber el primer caracter de un string

    • var primeraLetraNombre = nombre.charAt()

Atributos basicos

Saber la cantidad total de letras o caracteres de un string

  • var cantidadLetraNombre = nombre.length

Para actualizar el dato

Volver a declarar la variable

  • cantidadLetraNombre = nombre.length

Concatenacion

Esto lo que hace es poder unir dos variables tipo string

  • var nombreCompleto = nombre + ' ' + apellido

    • Interpolacion de text

    En esta sintaxis se puede colocar codigo js dentro de los parentesis lo que evita escribir una funcion aparte

  • var nombreCompletoInterpolacion = `${nombre} ${apellido.toUpperCase()}`
