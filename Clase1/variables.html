Variables en Javascript

Las variables tiene 3 grupos grandes los cuales son:

  • var: Es una variables global
  • const: Es una variable que no se puede cambiar su contendido como una variable de la velocidad de la luz
  • let: Es una variable que permite declarar limitando su alcance

Declarar variables

Para declar una variable string se asigna que tipo de variable sera, luego se coloca el nombre de la variable y posteriormente se asigan su valor en el caso de string se debe colocar comillas simple ' ' o comillas dobles " ".

var = nombre;
nombre = 'Israel';
Por convencion se recomienda usar comillas simples ' '.

El punto y coma ; es algo opcional ya que reconoce las lineas de codigo
